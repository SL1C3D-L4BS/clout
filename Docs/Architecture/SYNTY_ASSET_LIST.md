## Budget Strategy

Synty offers **SyntyPass** — a subscription at $30/month that unlocks the entire 130+ pack library. For a project of this scope, SyntyPass is overwhelmingly the most cost-effective path. Individual pack purchases would exceed $2,000+.

**Recommendation:** Subscribe to SyntyPass for the duration of active development. Download and integrate all needed packs. The subscription license covers commercial use.

## Integration Guide

### Import Order

1. **Starter Pack (FREE)** — Base reference for scale and style
2. **City Pack** — Primary environment, establishes road grid and building scale
3. **Gang Warfare Pack** — Test crime environment visual integration
4. **Modular City Characters** — Player character setup
5. **Police Station** — Law enforcement visual layer
6. **Heist Pack** — Crime target environments
7. **Palm City** — Waterfront district (import last among Tier 1 — largest pack)
8. **Remaining packs** — As needed per development phase

### Material Setup

All Synty POLYGON packs use a shared color palette system:

- Single texture atlas per pack with 4 color variations
- URP Lit shader (Synty provides URP-compatible shaders as of 2022.3+)
- Custom color shader allows runtime tinting
- All packs share the same art style — cross-pack mixing is seamless

### Character Rigging

- All characters use Unity Humanoid rig (Mecanim-compatible)
- Animations from any pack can be shared across all characters
- ThatAnimator's Souls-like anims retarget to Synty rigs via Humanoid mapping
- Modular character pieces parent to skeleton joints

### Scale Reference

- All POLYGON packs use consistent scale
- Characters are approximately 1.8 Unity units tall
- Doorways, vehicles, and props all match this scale
- No rescaling needed when mixing packs

### Performance Notes

- **GPU Instancing:** Enable on all Synty materials — they share textures, making instancing extremely effective
- **LOD Groups:** Synty packs include LOD levels for larger assets — ensure LOD Groups are configured
- **Static Batching:** Mark all non-moving environment geometry as static
- **Texture Atlas:** Synty's single-atlas approach is already optimized — don't break atlases
- **Occlusion Culling:** Critical for dense city environments — bake per district
- **Addressable Assets:** Use for district streaming — load/unload packs per area
